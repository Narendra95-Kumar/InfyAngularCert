Problem Description:

 

AntiqMart is an online showroom of antique items where customers across the globe can browse through the various antique items, view product details and also place an enquiry for the selected item.

 

Details of Backend API service (Implemented)-

  • Order Enquiry – This service registers enquiry details of the customers by sending POST request at /enquiries and it returns the following response:

    { "message": "Thanks for connecting with us. Your enquiry has been registered with enquiry id: 6. Please use this id during future contact." }

    •  

  • Fetch Antique Products – This service returns a JSON response containing the list of various antique products and their details by making a GET request to /products:

    You can hit http://localhost:3020/products from the browser and check the response.

    •  

  • Fetch Product Details – This service returns a JSON response containing details of the particular product with the given id by making a GET request to /products/:id:

    You can hit http://localhost:3020/products/3 from the browser and check the response.
